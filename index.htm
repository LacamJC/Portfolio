New NPR

Sobre

O projeto NPR (Natureza Prioridade Renovada) é uma aplicação web desenvolvida por um grupo de estudantes durante o meu curso técnico em 2023. Em certo ponto do curso, fui integrado ao grupo, e após isso, o desenvolvimento do projeto avançou além do esperado. Após o término do curso, continuei trabalhando nele e, após estudar sobre o Bootstrap 5, refiz completamente o projeto, visando um código mais limpo e fácil de entender. Renomeei-o como New NPR.

O que é

Sendo uma aplicação web que une a tecnologia e a reciclagem, criando uma plataforma que se propõe a mostrar pontos de coleta oficiais em determinados locais, exibindo endereço e tipo de coleta do local. A aplicação possui um sistema de cadastro para o usuário, proporcionando a oportunidade de criar novos pontos de coleta. O usuário envia um formulário com os dados necessários, e assim, o administrador pode adicioná-lo ao mapa como oficial.

PMG

Sobre

O PMG, sigla para Projeto Matemático Geral, foi uma ideia de concepção própria. Trata-se de uma plataforma que incorpora diversos algoritmos matemáticos, como diferentes tipos de calculadoras e outras fórmulas. Essa iniciativa visa proporcionar uma abordagem abrangente e inovadora para lidar com desafios matemáticos, oferecendo uma gama diversificada de ferramentas e recursos. O intuito é facilitar a compreensão e resolução de problemas, contribuindo para o desenvolvimento e aprimoramento das habilidades matemáticas.

O que é

Em sua essência, eram apenas alguns testes em JavaScript, mas com o passar do tempo, acabei desenvolvendo um projeto interessante e educativo. A proposta inicial de experimentação com código evoluiu para a modelagem de uma iniciativa que não só desperta interesse, mas também se revela educativa. Este projeto visa oferecer uma abordagem prática e envolvente, utilizando a linguagem de programação JavaScript como base, proporcionando uma experiência tanto educacional quanto desafiadora.
